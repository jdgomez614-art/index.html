Configuración
Obtén tu Client ID en Google Cloud Console:
1. Crear proyecto → Habilitar Drive API + Vision API
2. Credenciales → Crear ID de cliente OAuth 2.0 → Aplicación web
3. Agregar tu dominio de Vercel en "Orígenes JavaScript autorizados"
4. Pegar el Client ID aquí
Misma consola → Credenciales → Crear Clave de API
Habilita Cloud Vision API para este proyecto.
URL de Drive → drive.google.com/drive/folders/ESTE_ID
⚠️ Este modo solo funciona con carpetas públicas de Drive (compartidas con "Cualquier persona con el enlace").
Para carpetas privadas usa el modo OAuth2.
La carpeta debe ser pública. ID en: drive.google.com/drive/folders/ESTE_ID
